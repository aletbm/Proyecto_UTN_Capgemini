Sobre el proyecto

Este proyecto fue realizado con una finalidad educativa para la Academia Python brindada por la empresa Capgemini en colaboracion con la UTN FRBA, en el año 2022. El proyecto consiste en una pagina web que permite a los usuarios registrarse y loguearse, y poder jugar a contestar preguntas de distintos temas mediante la camara

Reconocimiento de la mano con OpenCV y MediaPipe

Este proyecto fue posible gracias a OpenCV, una libreria enfocada en la vision artificial en tiempo real. OpenCV es capaz de capturar frames directamente desde una camara, apartir de esto procedemos a trabajar sobre dicho frame con herramientas que la misma libreria nos proporciona, los diferentes procesamientos que podemos realizar sobre la imagen son diversos, entre ellos, redimensionamientos, cambios de escalas de color, difuminacion, deteccion de bordes, etc.
Una vez obtenidos y procesados estos frames con OpenCV, la libreria MediaPipe nos brinda las herramientas necesarias para percibir la forma y el movimiento de las manos. Esta libreria utiliza un pipeline de ML que consta de varios modelos que trabajan juntos. MediaPipe utiliza modelos de ML que detectan la palma de la mano y genera un corte preciso de la mano apartir de la posicion de esta, para posteriormente realizar una localizacion precisa de 21 puntos claves dentro de la mano. Finalmente MediaPipe nos entrega una lista que contiene informacion de la posicion de estos 21 puntos y si la mano es derecha o izquierda. Con estos datos fuimos capaces de detectar señas realizadas con las manos, tanto para la eleccion dentro de una lista de opciones de posibles respuestas a una pregunta como la determinacion final del envio de una respuesta.

Arquitectura de la plataforma con Django y MySQL

La arquitectura de este proyecto fue llevada a cabo con Django, un framework que respeta el patron de diseño modelo-vista-template (MVT) que facilita la creacion de sitios web complejos. Django es compatible con una variedad de sistemas de gestion de bases de datos, entre ellos, PostgreSQL, SQLite, Oracle, Microsoft SQL y MySQL, este ultimo sistema fue el utilizado para este proyecto debido a su velocidad y sencillez en su uso.
